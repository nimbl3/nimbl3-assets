# Nimble Assets Management

## Usage

Clone the repository

`git clone https://github.com/nimbl3/nimbl3-assets.git`

## Setup

- Requirements
  - Install Ganache
  - Install Metamask chrome extension

- Start the app

```
npm install
truffle compile
truffle migrate
npm run dev
```
